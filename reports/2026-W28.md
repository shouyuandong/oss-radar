- **Token 压缩重回焦点**：caveman（#1）以"穴居人简语"策略砍掉 65% token，是继 W23 headroom 之后又一个 Token 压缩项目登顶——Agent 的 token 经济学正从"压缩输入"演进到"压缩自身表达"。
- **安全赛道双项目入榜**：strix（#5, 连续两周）+ pentagi（#9, 新入）——渗透测试已从"辅助工具"升级为"全自动 Agent"，Go 实现的 pentagi 意味着企业级部署场景。
- **Agent 基础设施三连在榜**：orca（#11, 三周）、OmniRoute（#12, 三周）、herdr（#13, 三周）——并行调度/AI 网关/多路复用已成常驻基础设施层。
- **中国大厂密集出现**：alibaba/page-agent（#7↑8）、TencentCloud/CubeSandbox（#17）、iOfficeAI/OfficeCLI（#14）——阿里/腾讯/iOffice 均瞄准 Agent 执行环境。
- **MCP 持续渗透**：chrome-devtools-mcp（#3）、DesktopCommanderMCP（#19）、codex-plugin-cc（#16↑2）——MCP 协议从"工具桥接"扩展到"DevTools 暴露"和"跨 Agent 协作"。

- **设计体系 Agent Ready**：impeccable（#4, W23→W28 二度入榜）+ astryx（#18）——设计系统正在从"人用"转向"Agent 用"。
